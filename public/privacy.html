Política de Privacidade

Esta política é codificada no app e também disponível em /privacy no SPA.

1. Resumo

O Nexo Terminal é um jogo de investigação local. Não coletamos dados pessoais identificáveis dos jogadores em rotinas automáticas.

2. Dados coletados

  • LocalStorage: progresso de jogos, estado de casos e estatísticas básicas (crime_#).
  • Firebase: casos diários são carregados via Firestore/chamada HTTP; não armazena trabalho pessoal.
  • Sem cookies de terceiros e sem tracking de analytics por padrão.

3. Segurança

  • O app não envia dados de usuário ao servidor, exceto para leitura de casos diários.
  • Não há login ou senha exigidos.
  • O desenvolvedor não se responsabiliza por dados inseridos em campos de texto, pois o jogo não possui cadastros de usuário.

4. Direitos do usuário

  • O usuário pode limpar o histórico e o progresso via ferramentas do navegador (localStorage).
  • Pedido de exclusão total de dados remotos deve ser solicitado via canal de suporte se o serviço evoluir.

5. Contato

Reportar dúvidas ou incidentes de privacidade via README do projeto ou correio do repositório.

6. Atualizações

Esta política é parte do código e deve ser atualizada sempre que novas funcionalidades de coleta de dados forem adicionadas. Fechação: 29/03/2026.
